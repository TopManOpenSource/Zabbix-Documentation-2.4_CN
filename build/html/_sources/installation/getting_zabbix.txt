.. _获取zabbix:

*************
1.获取Zabbix
*************

概览
====

有三种途径获取Zabbix：

        # 从 `::ref:二进制包 <从二进制包安装>` 安装
        # 下载源码并且 `::ref:自行编译<编译安装>` 
        # 下载 `::ref:Zabbix容器<zabbix容器>` 

若要下载最新版的源码包或者zabbix容器，请访问 `::ref:Zabbix下载页面<http://www.zabbix.com/download.php>.` 这里提供最新版的直接下载链接，如想下载旧版本，请参阅下面的链接稳定版本下载。
