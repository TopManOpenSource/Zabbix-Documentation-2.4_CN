1.手册结构¶

结构¶

为了使用户能够快速访问到感兴趣的特定部分，Zabbix 2.4的手册划分为17章，每一章又包含了若干小节。当你使用导航栏访问这些章节时，请确保你展开了章节目录，以显示这些章节包含的完整内容。 手册提供了页面之间的链接，尽可能地保证用户不错过彼此有关联的内容。

章节目录及内容简介¶

介绍 一节提供了Zabbix当前版本的大概信息，阅读这一节会给你提供一些好的理由来选择Zabbix.

Zabbix基础概念 解释了Zabbix专业术语并且提供了Zabbix组件的详细信息。

安装与快速开始 帮助你开始使用Zabbix, Zabbix appliance则提供了另外一种快速开始Zabbix的选择。

配置 章节是本手册内容最多、非常重要的章节。包含了如何设置Zabbix来监视环境的高级要点、如何设置hosts,如何获取基本数据，如何查看基本数据，如何配置告警、如何在出现故障时远程执行命令。

IT-服务 一章详细说明了如何使用Zabbix对你的环境进行概览。

Web监视 章节教你如何监视Web站点的可用性。

监控虚拟机 展示了一个对VMware 环境的监视配置

维护、正则表达式、事件报告和XML导入导出 属于进阶章节，展示了如何使用Zabbix中的各个方面的功能。

自动发现 章节讲解了如何设置Zabbix在网络中自动发现网络设备、活动的客户端、主机、网卡等。

分布式监控 提供立刻在大而复杂的环境中使用Zabbix监视的方法。

Web接口 包含了使用Zabbix的网页接口时的特殊信息

API 章节介绍了ZabbixAPI的工作细节

详细的技术细节列表包含在附录中，你也可以在FAQ章节找到。
